# Walmart Sales Forecasting

## 📌 Project Overview
This project aims to forecast Walmart weekly store sales using data and external economic indicators. The goal is to build predictive models and generate actionable insights for retail decision-making.

## 🎯 Objectives
- Forecast weekly sales at store and department levels
- Identify key factors influencing sales
- Compare baseline and machine learning models
- Generate business insights

## 📂 Project Structure
- data/: raw and processed datasets
- notebooks/: analysis and modeling notebooks
- src/: reusable Python scripts
- outputs/: figures, models, and reports

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn, XGBoost
